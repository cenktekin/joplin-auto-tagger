# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq
- Translations: https://www.contributor-covenant.org/translations

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the maintainer at:

- Email: cenktekin@duck.com

All reports will be reviewed and investigated. The maintainers are obligated to respect the privacy and security of the reporter of any incident.
