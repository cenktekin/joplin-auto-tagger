# Security Policy

## Reporting a Vulnerability
- Email: cenktekin@duck.com
- Please include: affected version, environment, steps to reproduce, impact, and any PoC.
- Do not open public issues for security problems.

## Response Timeline
- Acknowledge receipt: within 72 hours
- Initial assessment: within 7 days
- Fix or mitigation target: within 30 days (complex issues may take longer)

## Supported Versions
We aim to support the latest tagged release and `main`. If a fix requires extensive changes, we may ship a new minor/patch release.

## Disclosure
We follow coordinated disclosure. We appreciate a reasonable time window to investigate and release a fix before public disclosure.

## Scope
- Joplin plugin code in this repository
- Build artifacts generated by this repo

Out of scope: upstream dependencies and Joplin core; please report those to their respective projects.
